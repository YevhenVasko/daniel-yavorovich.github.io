MySQL slow log

Tue 03 January 2012
By Даниэль Яворович

Для логгирования "долгих" запросов MySQL можно включить в секцию [mysqld] конфигурации MySQL (для RHEL/CentOS это /etc/my.cnf) следующие параметры

# Slow query log
long_query_time = 1
log-slow-queries = /var/log/mysql/slow_queries.log

  • long_query_time - значение в секундах, после которого запрос нужно считать "долгим" и записывать информацию о нём в log-файл

  • log-slow-queries - path к log-файлу MySQL, где будут фиксироваться запросы, что выполнялись дольше, чем это описано в long_query_time
